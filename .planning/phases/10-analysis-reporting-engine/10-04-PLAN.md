**report_generator.py** - `ReportGenerator` class:

Constructor takes:
- `config: AnalysisConfig | None = None`
- `template_dir: str | Path | None = None` (default: package templates/ directory)

Initializes Jinja2 Environment with FileSystemLoader pointing to template_dir.

Methods:

1. `def generate_markdown(self, synthesis: AnalysisSynthesis) -> str`:
   Render the intelligence_report.md.j2 template with synthesis data.
   - Extract all fields from AnalysisSynthesis for template context
   - Flatten snapshot data for evidence appendix (fact_id, claim_text, verification_status, etc.)
   - Render and return Markdown string

2. `def generate_executive_brief(self, synthesis: AnalysisSynthesis) -> str`:
   Render only the executive_brief.md.j2 template. Returns short Markdown string.

3. `async def save_markdown(self, markdown: str, output_path: str | Path) -> Path`:
   Write Markdown to file. Create parent dirs. Return Path. Use `asyncio.to_thread` for file I/O.

**pdf_renderer.py** - `PDFRenderer` class:

Constructor takes:
- `css_path: str | Path | None = None` (default: package styles/report.css)

Methods:

1. `async def render_pdf(self, markdown_content: str, output_path: str | Path) -> Path`:
   Convert Markdown to PDF:
   a. Parse Markdown to HTML via mistune: `html = mistune.html(markdown_content)`
   b. Wrap HTML in full document with CSS link
   c. Render to PDF via WeasyPrint: `HTML(string=full_html).write_pdf(output_path)` — run in `asyncio.to_thread()` to avoid blocking event loop (per RESEARCH.md anti-patterns)
   d. Return Path to PDF

   Wrap WeasyPrint call in try/except ImportError. If WeasyPrint not installed (system deps missing), log warning and return None. Caller handles gracefully.

2. `def _wrap_html(self, body_html: str) -> str`:
   Wrap body HTML in full document with CSS. Read CSS from css_path. Return complete HTML string with embedded `<style>` tag (avoids file path issues in WeasyPrint).

**report_store.py** - `ReportRecord` and `ReportStore`:

`ReportRecord(BaseModel)`:
- `investigation_id: str`
- `version: int` - Auto-incrementing version number
- `content_hash: str` - SHA256 of Markdown content for diff detection
- `markdown_content: str` - Full Markdown source
- `markdown_path: str | None = None` - Path to saved .md file
- `pdf_path: str | None = None` - Path to saved .pdf file
- `generated_at: datetime`
- `synthesis_summary: dict[str, Any] = Field(default_factory=dict)` - Summary stats (judgment count, confidence level, etc.)

`ReportStore` class:

Constructor takes:
- `output_dir: str = "reports/"`
- `persistence_path: str | None = None` (optional JSON persistence for report metadata)

In-memory storage: `dict[str, list[ReportRecord]]` keyed by investigation_id (list of versions, newest last).

Uses structlog, asyncio.Lock for thread safety.

Methods:

1. `async save_report(investigation_id: str, markdown_content: str, synthesis: AnalysisSynthesis | None = None, markdown_path: str | None = None, pdf_path: str | None = None) -> ReportRecord`:
   - Compute content_hash from markdown_content
   - Determine version: if investigation has existing reports, new version = last + 1. If content_hash matches latest version, skip (no change) and return existing record.
   - Create ReportRecord
   - Store in memory
   - Persist if persistence_path configured
   - Return the record

2. `async get_latest(investigation_id: str) -> ReportRecord | None`:
   Return the most recent version for the investigation.

3. `async get_version(investigation_id: str, version: int) -> ReportRecord | None`:
   Return specific version.

4. `async list_versions(investigation_id: str) -> list[ReportRecord]`:
   Return all versions for an investigation, ordered by version number.

5. `async has_changed(investigation_id: str, markdown_content: str) -> bool`:
   Compare content_hash of new content vs latest stored version. Returns True if different (or no previous version exists).

6. `async list_investigations(self) -> list[dict[str, Any]]`:
   Return summary of all investigations with report counts and latest version info.

7. `_compute_hash(content: str) -> str`:
   SHA256 of content string.
