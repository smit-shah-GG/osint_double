  2. **Add Cloudflare challenge detection function** (module-level, before class definitions):
     ```python
     _CF_CHALLENGE_INDICATORS = [
         re.compile(r'cf-turnstile', re.IGNORECASE),
         re.compile(r'challenge-form', re.IGNORECASE),
         re.compile(r'__cf_chl_f_tk', re.IGNORECASE),
         re.compile(r'cf-browser-verification', re.IGNORECASE),
         re.compile(r'cf_clearance', re.IGNORECASE),
         re.compile(r'Checking your browser', re.IGNORECASE),
         re.compile(r'Enable JavaScript and cookies to continue', re.IGNORECASE),
         re.compile(r'ray\s+id', re.IGNORECASE),
     ]

     def is_cloudflare_challenge(html: str) -> bool:
         """Detect Cloudflare challenge/interstitial pages. Requires 2+ indicators."""
         if len(html) > 50000:
             return False
         hits = sum(1 for p in _CF_CHALLENGE_INDICATORS if p.search(html[:5000]))
         return hits >= 2
     ```

  3. **Add BrowserPool class** (before HybridWebCrawler):
     - `__init__(self, max_contexts: int = 5)`: asyncio.Semaphore(max_contexts), _playwright=None, _browser=None
     - `async start(self) -> None`: Launch playwright + chromium with `--disable-blink-features=AutomationControlled`
     - `async fetch(self, url: str, user_agent: str, timeout_ms: int) -> str`:
       - Acquire semaphore
       - Create new_context with user_agent, viewport 1920x1080, locale="en-US", timezone_id="America/New_York"
       - Create new_page
       - Apply `await stealth_async(page)` (import from playwright_stealth)
       - `page.goto(url, timeout=timeout_ms, wait_until="networkidle")`
       - `html = await page.content()`
       - Check `is_cloudflare_challenge(html)` -- if True, log warning and return empty string
       - **CRITICAL**: `finally: await context.close()` -- prevents context leak
       - Return html
     - `async stop(self) -> None`: Close browser, stop playwright
     - `@property started(self) -> bool`: Return self._browser is not None

  4. **Modify HybridWebCrawler** to use BrowserPool:
     - Add `_browser_pool: Optional[BrowserPool] = None` attribute in __init__
     - Add `async def _ensure_browser_pool(self) -> BrowserPool`: lazy-start pattern, creates and starts pool on first call
     - **Replace `_playwright_fetch`**: Instead of `async with async_playwright() as p: browser = await p.chromium.launch(...)`, use `pool = await self._ensure_browser_pool(); html = await pool.fetch(url, self._get_user_agent(), int(self.playwright_timeout * 1000))`
     - If html is empty string (Cloudflare challenge detected), return failure result with error "Cloudflare challenge page detected"
     - Add `async def close(self)` override that also stops the BrowserPool if started
     - Keep the existing httpx-first hybrid logic intact -- only the Playwright fallback path changes
