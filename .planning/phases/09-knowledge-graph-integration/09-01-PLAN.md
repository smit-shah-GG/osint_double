  <action>
Create `osint_system/data_management/graph/` package.

**schema.py** - Define the following Pydantic models:

1. `EdgeType(str, Enum)` with ~12-15 semantic relationship types per CONTEXT.md:
   CORROBORATES, CONTRADICTS, SUPERSEDES, MENTIONS, SOURCED_FROM, PART_OF, CAUSES, PRECEDES, LOCATED_AT, ATTRIBUTED_TO, RELATED_TO, VERIFIED_BY, HAS_CLASSIFICATION. Group them by category (structural, semantic, temporal, verification) with docstrings explaining when each is used.

2. `GraphNode(BaseModel)`:
   - `id: str` (unique node key, format: `{label}:{natural_key}` e.g. `Fact:fact-uuid-123`)
   - `label: str` (node type: Fact, Entity, Source, Investigation, Classification)
   - `properties: dict[str, Any]` (all node properties)
   - Helper property methods: `investigation_id`, `name_or_id`

3. `GraphEdge(BaseModel)`:
   - `source_id: str` (from node ID)
   - `target_id: str` (to node ID)
   - `edge_type: EdgeType`
   - `weight: float = Field(default=0.5, ge=0.0, le=1.0)`
   - `properties: dict[str, Any]` (metadata: timestamp, evidence_count, authority, source)
   - `cross_investigation: bool = False` (per CONTEXT.md: flagged, not auto-trusted)

4. `QueryResult(BaseModel)`:
   - `nodes: list[GraphNode]`
   - `edges: list[GraphEdge]`
   - `query_type: str` (entity_network, corroboration_clusters, timeline, shortest_path, raw)
   - `metadata: dict[str, Any]` (query params, timing, counts)
   - Helper methods: `node_count`, `edge_count`, `to_dict()`

5. `compute_edge_weight(evidence_count: int, authority_score: float, recency_days: int, base_weight: float = 0.5) -> float`:
   Per RESEARCH.md formula: `base + authority_score*0.3 + min(0.2, 0.05*log1p(count)) - min(0.2, days/365*0.2)`, capped [0.0, 1.0].

**adapter.py** - Define `GraphAdapter` as a `typing.Protocol` with `@runtime_checkable`:

Methods (all async):
- `initialize() -> None` - Create indexes/constraints/schema
- `close() -> None` - Clean shutdown
- `merge_node(label: str, properties: dict, key_property: str = "id") -> str` - Returns node ID
- `merge_relationship(from_id: str, to_id: str, rel_type: str, properties: dict) -> None`
- `batch_merge_nodes(label: str, nodes: list[dict], key_property: str = "id") -> int` - Returns count
- `batch_merge_relationships(relationships: list[dict]) -> int`
- `delete_node(node_id: str) -> bool`
- `query_entity_network(entity_id: str, max_hops: int = 2, investigation_id: str | None = None) -> QueryResult`
- `query_corroboration_clusters(investigation_id: str) -> QueryResult`
- `query_timeline(entity_id: str, investigation_id: str | None = None) -> QueryResult`
- `query_shortest_path(from_entity_id: str, to_entity_id: str, investigation_id: str | None = None) -> QueryResult`
- `execute_cypher(query: str, parameters: dict | None = None) -> list[dict]`

Each method must have a full docstring. Import QueryResult from schema.py.

**__init__.py** - Export GraphAdapter, GraphNode, GraphEdge, QueryResult, EdgeType, compute_edge_weight.

All models use `model_config` with examples like existing schemas do.

Create `osint_system/config/graph_config.py` following the pattern in `settings.py`:

1. `GraphConfig(BaseModel)` with fields:
   - `neo4j_uri: str = Field(default="bolt://localhost:7687")` - from `NEO4J_URI` env var
   - `neo4j_user: str = Field(default="neo4j")` - from `NEO4J_USER` env var
   - `neo4j_password: str = Field(default="osint_dev_password")` - from `NEO4J_PASSWORD` env var
   - `neo4j_database: str = Field(default="neo4j")` - from `NEO4J_DATABASE` env var
   - `use_networkx_fallback: bool = Field(default=False)` - from `GRAPH_USE_NETWORKX` env var, True forces NetworkX even if Neo4j available
   - `batch_size: int = Field(default=5000, ge=100, le=50000)` - UNWIND batch size
   - `max_hops: int = Field(default=3, ge=1, le=10)` - Default max hops for path queries
   - `llm_relationship_extraction: bool = Field(default=False)` - from `GRAPH_LLM_EXTRACTION` env var, gates LLM cost per RESEARCH.md open question 4
   - `cross_investigation_matching: bool = Field(default=True)` - Enable cross-investigation entity matching

2. Class method `from_env()` that reads from environment variables using `os.getenv()`, consistent with how `settings.py` works. Use `python-dotenv` `load_dotenv()` if `.env` file exists.

3. Update `.env.example` (or the `.env` file) to add Neo4j env vars:
   ```
   # Neo4j Graph Database Configuration
   NEO4J_URI=bolt://localhost:7687
   NEO4J_USER=neo4j
   NEO4J_PASSWORD=osint_dev_password
   NEO4J_DATABASE=neo4j
   GRAPH_USE_NETWORKX=false
   GRAPH_LLM_EXTRACTION=false
   ```

Do NOT import or depend on neo4j driver here -- this is pure config.
